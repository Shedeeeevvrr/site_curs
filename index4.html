которые будут правильно работать на всех доступных устройствах,
которые есть у ваших пользователей.

Адаптивная вёрстка с медиа-запросами в HTML и CSS
1. Что такое адаптивная вёрстка?
Адаптивная вёрстка — это метод создания веб-страниц,
которые корректно отображаются на разных устройствах и экранах (ПК, планшеты, смартфоны).
Основной инструмент для этого в CSS — медиа-запросы (media queries).

2. Основы медиа-запросов
Медиа-запросы позволяют применять CSS-правила в зависимости от характеристик устройства.
Например:
- ширины и высоты экрана
- разрешения дисплея
- ориентации экрана (альбомная или портретная)
- типов устройств (экран, принтер и т. д.)
Пример базового медиа-запроса:
@media (max-width: 768px) { body { background-color: lightgray; } }
Этот код меняет цвет фона на светло-серый, если ширина экрана 768 пикселей или меньше.

3. Принципы адаптивного дизайна
- Мобильный в первую очередь (Mobile First) — сначала стили разрабатываются для мобильных
устройств, а затем расширяются для больших экранов.
- Гибкие сетки (Flexible Grid) — использование
display: flex;, display: grid;.
- Гибкие изображения (Responsive Images) — max-width: 100%
для предотвращения выхода за границы контейнера.
- Относительные единицы измерения — использование
%, em, rem, vw, vh вместо фиксированных px.

4. Основные брейкпоинты (breakpoints)
Общие брейкпоинты для адаптивной вёрстки:
/* Смартфоны (до 600px) */ @media (max-width: 600px) { ... }
/* Планшеты (601px - 1024px) */ @media (min-width: 601px) and (max-width: 1024px) { ... }
/* Десктопы (от 1025px) */ @media (min-width: 1025px) { ... }

5. Использование flexbox и grid
Flexbox для адаптивного расположения элементов:
.container {
display: flex;
flex-wrap: wrap;
justify-content: space-between;
}
.item {
width: 45%;
margin-bottom: 20px;
}
@media (max-width: 600px) {
.item { width: 100%; } }
Grid для создания адаптивных сеток
.container {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
gap: 20px;
}

6. Частые ошибки и проблемы
Использование px вместо %, vw, em Игнорирование viewport meta в <head>:
<meta name="viewport" content="width=device-width, initial-scale=1">
Не тестировать сайт на разных устройствах и эмуляторах.
Адаптивная вёрстка — важный навык для создания удобных и доступных сайтов.
Используйте медиа-запросы и современные CSS-технологии для лучшей оптимизации!
